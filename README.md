# 💸 쉽고 빠른 간편 결제, 자동 결제 서비스 - 즉시페이

## 📋 프로젝트 설명

- 즉시페이를 통해 결제 기능이 필요한 모든 서비스에 결제 기능 제공
- 멤버십 서비스, 구독 서비스 등 일정 주기로 결제가 필요한 서비스를 위해 자동 결제 기능 제공
- MSA(Micro Service Architecture)
- Hexagonal Architecture
